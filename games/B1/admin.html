🛠️ 개발자 페이지

게임 배정 허브 — 누구에게 / 어떤 게임을 / 어떤 모드로 돌릴지 정한다

새 게임 배정

앞으로 B2, B3 게임세트가 추가되면 여기에 나타납니다.
양식: 반,번호,이름 (반 명단 탭의 샘플과 동일). 이 배정에만 쓰입니다.

👥 반 명단 등록 (상비)

학기 내내 쓰는 반 명단을 등록해 두면, 로그인 모드 배정 시 학생이 짝 이름을 명단에서 고릅니다. (즉석 그룹은 배정 탭에서 그때 올릴 수 있어요)
⚠️ 학생 실명입니다. 이 PC에만 저장되며, 공개되지 않습니다.
예: 2반,1,김민준 · 샘플 CSV 받기
시트를 "링크가 있는 모든 사용자 — 뷰어"로 공개한 뒤 주소를 붙여넣으세요.

🧪 테스트 도구

게임이 잘 도는지 점검하는 도구입니다. (학생에게 배포하는 화면이 아닙니다)
🎮 플레이 시뮬카메라 없이 게임 걸어보기 (테스트 모드) 🧪 분산 시뮬레이터가상 조로 혼잡 배정 검증

🛠 준비 도구

게임을 만들기 전에 쓰는 도구입니다.
🎯 출제 빌더문제·정답·힌트·이미지 편집 🔳 QR 인쇄장소별 QR코드 출력
⚠️ 이 개발자 페이지는 운영자 전용입니다. 학생에게는 배정 이력에서 만든 QR/링크만 주세요.
