# 🏕️ 문제 초안 풀 v1 — 검수·선별용

## 🎯 이 초안의 설계 원칙

본격 제작 전에 합의했던 4가지 철학을 그대로 반영했습니다.

| 원칙 | 적용 |
|------|------|
| **검색으로 안 풀리게** | 폰 들고 있는 학생이 구글에 쳐도 안 나오는 **추론·넌센스·현장 관찰·협력 분해** 위주 |
| **LLM은 각색기** | 단순 생성 대신 캠핑 테마로 각색, 난이도 조정, 힌트 3단계 |
| **8장소 QR 유형 믹스** | 공통 3 + 2글자 2 + 3글자 2 + 4글자 1 |
| **멀티미디어 데모** | 이미지·유튜브·오디오 문제 각 1개씩 포함 (Phase B2 렌더러 확장 대상) |

### 협력형 설계 두 방식 (★ 골라주세요)

| 방식 | 작동 | 협력 강도 | 예시 |
|------|------|----------|------|
| **A. 떠올리기형** | 각자 같은 단어를 다른 힌트로 유도 → 글자 추출 | 중 (빠른 공유가 핵심) | 기존 "무지개" 방식 |
| **B. 독립 조합형** | 각 글자가 **서로 다른 미니퀴즈**의 답. 합쳐야 비로소 의미 | 강 (혼자선 단어 못 봄) | 아래 "손전등"·"보물상자" |

> 💬 **제 추천:** 가능한 곳은 **B(독립 조합형)** 가 게임성이 훨씬 좋습니다. 혼자선 합본 단어를 절대 못 보거든요. 다만 모든 단어에 B가 되진 않아서, 안 되는 곳은 A로 갔습니다. 각 미션에 `[방식 A]`/`[방식 B]` 표기해뒀어요.

- **설계의도:** 검색 무의미, 순수 추론. 한글 자음 순서만 알면 풀림 → 워밍업으로 적당.

## 🧩 최종 미션: 글자 조합 (★ 확정)

8미션을 풀면 합본 답들이 **탐험 도구**(`지도 · 손전등 · 보물상자 · 탐험가`)로 이어지는 게 1차 스토리. 그 위에 **마지막 미션 = 흩어진 8글자 조합**을 얹습니다.

### 작동 방식 — 무순서 수집 → 순수 조합

각 미션을 풀면 **암호 글자 1개**(문제 답과 별개인 수집 조각)를 받습니다. **자리·순서 표시 없이** 그냥 모읍니다. 8개째 미션을 푸는 순간(조마다 그 시점은 다름) **최종 조합 미션**이 발동돼요.

| set | 1 | 2 | 3 | 4 | 5 | 6 | 7 | 8 |
|-----|---|---|---|---|---|---|---|---|
| **암호 글자(예시)** | 운 | 동 | 장 | 큰 | 은 | 행 | 나 | 무 |

> ⚠️ 위 set→글자 배정은 **무순서 수집**이라 어떤 미션이 어떤 글자를 주든 상관없습니다. (자리 의미 X)

### 🔑 기본 예시 정답: `운동장큰은행나무`

흩어진 8글자에서 익숙한 **"운동장"** + **"은행나무"** 가 보이면 조합 성공. **힌트 없이도** 친숙한 단어라 풀 만합니다. → 선생님은 운동장 큰 은행나무 아래(또는 근처)에 실제 보물을 숨김.

### 화면 흐름

```
미션 풀 때마다 → 암호 글자 1개 획득 (자리·순서 없이 수집함에 쌓임)
   ↓ (8개째 미션 풀면 = 어느 set이든 그 조의 마지막)
🧩 최종 조합 미션 발동
   → 흩어진 8글자 제시 (힌트 없음)
   → 조원이 머리 맞대 장소명 조합 → 입력
   ↓ 정답이면 (normalizeAnswer: 공백 무시 비교)
🏆 done.html: "보물은 거기 있다!" → 선생님이 숨긴 실제 보물로
```
