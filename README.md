## 🎮 게임 방식

- 조를 나눠 각 조가 **정해진 순서**대로 학교 곳곳의 미션 장소를 돌아다닙니다.
- 각 장소에 QR코드가 1~4개 붙어 있고, 조원들이 나눠 스캔해서 문제를 풉니다.
- **모든 미션 장소를 가장 빨리 완주한 조가 우승!**

### 핵심 특징

| 특징 | 설명 |
|------|------|
| **조별 다른 순서** | 각 조가 서로 다른 장소 순서로 이동 → 한 곳에 몰리지 않음 |
| **커닝 방지** | 같은 장소라도 조마다 다른 미션 번호 표시 → "몇 번 답 뭐야?"가 통하지 않음 |
| **진행 복구** | 폰이 꺼지거나 새로고침해도 이어서 진행 |
| **이미지 문제** | 사진/그림이 포함된 문제 지원 |

## 🔳 장소별 QR 유형

장소마다 QR코드 개수에 따라 게임 방식이 달라집니다.

### QR 1개 — 공통 문제
```
조원 전체가 같은 QR을 스캔
   ↓
모두 같은 문제 (어려운 문제)
   ↓
한 명이 먼저 풀면 → 정답이 화면에 크게 표시됨
   ↓
"친구들에게 정답을 알려주세요!" → 각자 폰에 입력 → 통과
```
> 💡 **포인트:** 잘하는 학생이 풀어서 조를 빠르게 이끌 수 있어요(캐리).

---

### QR 2개 — 2글자 협력 조합
```
A 스캔한 학생: 국어 문제 → 답 1글자 (예: "사")
B 스캔한 학생: 수학 문제 → 답 1글자 (예: "과")
   ↓
서로 답을 알려주고 → 각자 "사과" 입력 → 통과
```

### QR 3개 — 3글자 협력 조합
```
A → "무" / B → "지" / C → "개"  →  각자 "무지개" 입력
```

### QR 4개 — 4글자 협력 조합
```
A → "나" / B → "무" / C → "그" / D → "늘"  →  각자 "나무그늘" 입력
```

> 💡 **포인트:** 조합 문제는 **혼자서는 절대 못 풀어요.** 반드시 함께 소통해야 정답이 완성됩니다.

## 📱 플레이 흐름

### 1단계 — 시작 (조장 폰에서)
1. https://betterwithu.github.io/school-explorer/ 접속
2. **우리 조 이름** 선택 (1조~4조)
3. **첫 번째 미션 장소** 안내가 뜸 → 그 장소로 이동

### 2단계 — 미션 풀기 (반복)
1. 장소에 도착 → QR코드를 각자 스캔
   - QR 1개면: 모두 같은 QR 스캔
   - QR 여러 개면: 각자 다른 QR 하나씩 선택
2. 화면에 문제 등장
3. 필요하면 힌트 보기 (최대 3개)
4. 정답 입력 → **정답이면 다음 장소 안내**
   - 공통 문제: 정답이 화면에 표시됨 → 친구들에게 공유

### 3단계 — 완주
- 모든 미션 장소를 완료하면 🏆 **완료 화면** 등장
- 가장 빨리 완주한 조가 우승!

## 📝 문제 출제 방법 (스프레드시트)

**스프레드시트 주소:** (담당 선생님께 문의)

### [문제] 탭 입력 방법

| 컬럼 | 설명 | 예시 |
|------|------|------|
| **id** | 문제 고유번호. 조합이면 `1A`, `1B` / 공통이면 `1` | `1A` |
| **세트** | 장소 번호. 같은 세트가 한 장소에 배치됨 | `1` |
| **순번** | 조합 문제면 `A`, `B`, `C`, `D` 순서. 공통이면 **비워두기** | `A` |
| **분류** | 과목 | `수학`, `국어`, `과학` |
| **난이도** | 드롭다운 선택 | `쉬움` / `보통` / `어려움` |
| **문제** | 문제 내용 | `캠핑장에 텐트 3개...` |
| **이미지** | 파일명 또는 URL (선택) | `tent.jpg` |
| **정답** | 조합이면 1글자, 공통이면 정답 전체 | `사` |
| **힌트1~3** | 점점 구체적으로 (선택) | `나무에서 열려요` |

### 입력 예시

**공통 문제 (QR 1개):**
| id | 세트 | 순번 | 분류 | 난이도 | 문제 | 정답 |
|----|------|------|------|--------|------|------|
| 5 | 5 | (비움) | 수학 | 어려움 | 6×7은? | 42 |

**2글자 조합 (QR 2개):**
| id | 세트 | 순번 | 분류 | 문제 | 정답 |
|----|------|------|------|------|------|
| 1A | 1 | A | 국어 | 과일 이름 첫 글자? | 사 |
| 1B | 1 | B | 수학 | 1+1을 한자로? | 과 |
→ 합본 정답: **사과** (A+B 자동 계산)

### [코스] 탭 — 장소 배치

| 세트 | 장소이름 |
|------|---------|
| 1 | 운동장 미끄럼틀 |
| 2 | 급식실 앞 |
| 3 | 도서관 입구 |

### 이미지 문제 사용하기
1. 이미지 파일을 repo의 `images/` 폴더에 추가
2. [문제] 탭 "이미지" 칸에 파일명만 입력 (예: `tent.jpg`)
3. 웹 주소를 직접 입력해도 됩니다 (예: `https://...`)

## 🚀 사이트에 반영하기 (GAS)

문제 입력 후 스프레드시트 상단 메뉴에서:

```
🏕️ 탐험대 → 사이트에 반영하기
```

1~2분 후 사이트에 자동 반영됩니다.

| 메뉴 | 기능 |
|------|------|
| 시트 양식 만들기 | 처음 사용 시 탭·헤더 자동 생성 |
| 사이트에 반영하기 | 시트 → GitHub 자동 커밋 → 사이트 갱신 |
| 미리보기 (반영 안 함) | 변환 결과만 확인 |
| 설정값 확인 | GitHub 토큰 등록 상태 확인 |

## 🖨️ QR코드 인쇄하기

1. https://betterwithu.github.io/school-explorer/qrcodes.html 접속
2. 장소별 QR코드 자동 생성됨
3. **🖨️ QR코드 인쇄하기** 버튼 클릭
4. 각 QR 아래에 장소 이름이 표시됨
   - QR 2개 이상인 장소: `1A`, `1B` 처럼 구분 표시
   - QR 1개(공통): `공통` 표시
5. 인쇄 후 해당 장소에 부착

> 💡 **부착 팁:** 비닐 봉투나 라미네이팅으로 보호하면 야외에서도 오래 써요.

## ❓ 자주 묻는 질문

**Q. 인터넷이 없는 환경에서도 되나요?**
A. 처음 접속 시 인터넷이 필요합니다. 접속 후에는 기본적인 진행이 가능하지만, 페이지 이동 시마다 인터넷이 필요합니다.

**Q. 폰을 바꿔도 이어서 할 수 있나요?**
A. 같은 조 이름으로 시작하면 진행 순서는 동일하지만, 진행 기록(localStorage)은 기기에 저장돼 다른 폰에서 이어하기는 안 됩니다. 같은 폰으로 계속 진행해주세요.

**Q. 조가 4개 이상이면 어떻게 하나요?**
A. 스프레드시트 [설정] 탭의 `teams` 값을 수정하세요. 예: `1조,2조,3조,4조,5조`

**Q. 미션 장소 수를 바꾸고 싶으면?**
A. 스프레드시트 [코스] 탭에서 장소를 추가/삭제하고 "사이트에 반영하기"를 누르면 됩니다.
